# Gaussian Splat Output Folder

Place your trained Gaussian Splat file here for the blog viewer embed.

Bundled sample paths:

- `media/gaussian-splats/output/train.splat`
- `media/gaussian-splats/output/example-nike.splat`

Your trained output path:

- `media/gaussian-splats/output/scene.ply`

After you generate the file and refresh the post page, set `data-splat` to `scene.ply` to view your own result.
